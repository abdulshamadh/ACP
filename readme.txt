--------------------------------------------------------------------------------------------------------
How to install the this application 
--------------------------------------------------------------------------------------------------------

Step 1 : Please copy the following folder "Files/SourceCode/" and paste to your local folder. (i.e. xampp\htdocs)

Step 2 : Please import the sql dump data from the following folder "Files/SQL Dump/technicalexam.sql" to your local phpmyadmin.

Step 3 : Please change the database details from the following folder "Files/SourceCode/class/config.ini".

----------
Example : 
----------

HOST=localhost
USERNAME=root	
PASSWORD=
DATABASE=technicalexam
